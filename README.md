# LinkLens

## 수동 점검 시나리오 (인증)

1. **정상 로그인**
   - 유효한 이메일/비밀번호 입력 후 로그인한다.
   - 인증 화면이 닫히고 사용자 이메일/아바타가 표시되는지 확인한다.
   - 링크/컬렉션 데이터가 정상 로드되는지 확인한다.

2. **잘못된 비밀번호**
   - 가입된 이메일에 잘못된 비밀번호를 입력해 로그인한다.
   - `이메일 또는 비밀번호가 올바르지 않습니다` 오류 문구가 표시되는지 확인한다.

3. **Supabase SDK 미로딩 graceful fallback**
   - DevTools에서 Supabase CDN 스크립트를 차단하거나 실패시키고 새로고침한다.
   - 인증 관련 동작 시 SDK 미로딩 안내 문구가 표시되고 화면이 깨지지 않는지 확인한다.

4. **기존 세션 복구**
   - 로그인 후 페이지를 새로고침한다.
   - 저장된 세션으로 자동 로그인되고, 인증 화면 없이 앱이 표시되는지 확인한다.
   - 세션 조회 실패 시 인증 화면과 `서비스 초기화 실패, 새로고침 후 재시도` 문구가 표시되는지 확인한다.
